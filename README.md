# Ula_4bits
## O grupo é composto pelos integrantes: André Giron e Ruth Menezes.

## Operações
|Seletor| Operação|
|---|---|
| `000` | Soma |
| `001` | Substração_2 |
| `010` | Complemento_2|
| `011` | Incremento|
| `100` | Troca de sinal|
| `101` | Shift |
| `110` | Dobrar |
| `111` | Comparar |

## Significados do Array de flags
|Endereço| Significado|
|---|---|
| '00' | Zero |
| '01' | Sinal |
| '10' | Carry out |
| '11' | Overflow |
